TUTORIAL JAVASCRIPT - Funciones Clásicas

Invocación, tipos, objetos (arguments y this), creación (bind...)

Invocación

El código dentro de una función no se ejecuta cuando se define la función, sino cuando se invoca.

Ejemplo:

function multiplicar(a, b) {
    return a * b;
}

multiplicar(10, 2);

arguments

Es un objeto similar a Array accesible dentro de funciones que contiene los valores de los argumentos pasados a esa función.
Solo funciona en las funciones clásicas, no arrow.

Ejemplo:

function test(a, b, c) {
    console.log(arguments[0]);
    // Salida: 1

    console.log(arguments[1]);
    // Salida: 2

    console.log(arguments[2]);
    // Salida: 3
}

test(1, 2, 3);

this

El valor de this está determinado por cómo se invoca a la función. No puede ser establecida mediante una asignación en tiempo de ejecución, y puede ser diferente cada vez que la función es invocada

Ejemplo:

let persona = {
    nombre: 'Paco',
    edad: 22,
    hablar: function () {
        console.log(this.nombre);
    }
};

persona.hablar(); // Paco

bind

El método bind() crea una nueva función, que cuando es llamada, asigna a su operador this el valor entregado, con una secuencia de argumentos dados precediendo a cualquiera entregados cuando la función es llamada.
El valor de this es ignorado cuando la función es llamada con el operador new.

Ejemplo:

let persona = {
    nombre: 'Antonio',
    edad: 25,
}

let logNombre = function(lang1, lang2) {
    console.log('INFO: ' + this.nombre + ', ' +this.edad)
    console.log('ARGS: ' + lang1 + ' ' + lang2)
}

let logNombrePersona = logNombre.bind(persona)
logNombrePersona('en', 'es')

// Salida: INFO: Antonio, 25
// Salida: ARGS: en es

apply

Invoca una determinada función asignando explícitamente el objeto this y un array o similar como parámetros (argumentos) para dicha función.

Ejemplo:

const persona = {
    nombreCompleto: function() {
        return this.nombre + " " + this.apellidos;
    }
}

const objPersona = {
    nombre: "Daniel",
    apellidos: "Ayala Cantador"
}

persona.nombreCompleto.apply(objPersona);
// Salida: "Daniel Ayala Cantador"

call

Llama a una función con un valor dado this y con argumentos provistos individualmente..

Ejemplo:

function sumar(x, y) {
    return x + y;
}

let resultado = sumar.call(this, 10, 20);
console.log(resultado); 
// Salida: 30

IIFE

(Immediately Invoked Function Expression) es una función de JavaScript que se ejecuta tan pronto como se define.

Es un patrón de diseño también conocido cómo función autoejecutable (Self-Executing Anonymous Function) y se compone por dos partes. La primera es la función anónima con alcance léxico encerrado por el Operador de Agrupación (). Esto impide accesar variables fuera del IIFE, así cómo contaminar el alcance (scope) global.

La segunda parte crea la expresión de función cuya ejecución es inmediata (), siendo interpretado directamente en el engine de JavaScript.

Toda declaración dentro del IIFE no será visible fuera de él.

Ejemplo:

(() => {
    // Iniciación de variables
    let clase;
    let comida;
})();

// Clase y comida no existen después de la ejecución de la función.
