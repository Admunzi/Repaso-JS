TUTORIAL JAVASCRIPT - Arrays

Every, forEach, some, slice, pop, push, find, map

every

Determina si todos los elementos en el array satisfacen una condición.

Ejemplo:

const esMenorDe40 = (valorActual) => valorActual < 40;

const lista = [1, 30, 39, 29, 10, 13];

console.log(lista.every(esMenorDe40));
// Salida: true

foreach

Ejecuta la función indicada una vez por cada elemento del array.

Ejemplo:

const lista = ['a', 'b', 'c'];

lista.forEach(elemento => console.log(elemento));

// Salida: "a"
// Salida: "b"
// Salida: "c"

some

Comprueba si al menos un elemento del array cumple con la condición implementada por la función proporcionada.

Ejemplo:

function esMayorDe5(elemento, index, array) {
    return elemento > 5;
}

console.log([-2, 5, 3, 1, 8].some(esMayorDe5)); 

// Salida: true

slice

Devuelve una copia de una parte del array dentro de un nuevo array empezando por inicio hasta fin (fin no incluido). El array original no se modificará.

Ejemplo:

let nombres = ['Rita', 'Pedro', 'Miguel', 'Ana', 'Vanesa'];
let masculinos = nombres.slice(1, 3);

// masculinos contiene ['Pedro','Miguel']

pop

Elimina el último elemento de un array y lo devuelve. Este método cambia la longitud del array.

Ejemplo:

const lista = ['patata', 'lechuga', 'pepino', 'lenteja', 'tomate'];

console.log(lista.pop());
// Salida: "tomate"

console.log(lista);
// Salida: Array ['patata', 'lechuga', 'pepino', 'lenteja']

push

Añade uno o más elementos al final de un array y devuelve la nueva longitud del array.

Ejemplo:

const animales = ['cerdo', 'pato', 'cabra'];

animales.push('vaca');

// Salida: Array ['cerdo', 'pato', 'cabra', 'vaca']

find

Devuelve el valor del primer elemento del array que cumple la función de prueba proporcionada.

Ejemplo:

const lista = [5, 12, 8, 130, 44];

const encontrado = lista.find(element => element > 10);

console.log(encontrado);
// Salida: 12

map

Crea un nuevo array con los resultados de la llamada a la función indicada aplicados a cada uno de sus elementos.

Ejemplo:

let lista = [1, 5, 10, 15];

let dobles = lista.map(function(x) {
    return x * 2;
});

// lista mantiene [1, 5, 10, 15]
// dobles ahora [2, 10, 20, 30]
