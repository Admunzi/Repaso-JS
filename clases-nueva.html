TUTORIAL JAVASCRIPT - Clases: Sintaxis Nueva

Constructor, set, get, privacidad, método, estático

Constructor

Ejemplo:

class Rectangle {
    height = 0;
    constructor(height) {
        this.height = height;
    }
}
let rect1 = new Rectangle(20);

Set

Ejemplo:

class Form{
    set foo(val){
        this.fooValue = val;
    }
}

let frm = new Form();
frm.foo = "bar";

Get

Ejemplo:

class Form{
    get foo(){
        return this.fooValue;
    }
}

let frm = new Form();
console.log(frm.foo);

Privacidad

Para que una propiedad o metodo sea privado solo se tiene que añadir # antes

Ejemplo:

class ClaseConPropiedadPrivada {
    #propiedadPrivada;
}

class ClaseConMetodoPrivado {
    #metodoPrivado() {
        return 'hola mundo';
    }
}

class ClaseConPropiedadEstaticaPrivada {
    static #PROPIEDAD_ESTATICA_PRIVADA;
}

class ClaseConmetodoEstaticoPrivado {
    static #metodoEstaticoPrivado() {
        return 'hola mundo';
    }
}

Métodos

Ejemplo:

class Form{
    saluda () {

console.log(frm.saluda());
