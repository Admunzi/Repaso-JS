TUTORIAL JAVASCRIPT - Funciones

Invocación, tipos, objetos (arguments y this), creación (bind...), arrow (lo más compactas)

Invocación

El código dentro de una función no se ejecuta cuando se define la función.
El código dentro de una función se ejecuta cuando se invoca la función.

Ejemplo:

function myFunction(a, b) {
    return a * b;
}

myFunction(10, 2);

Arguments

Es un objeto similar a Array accesible dentro de funciones que contiene los valores de los argumentos pasados a esa función.

Ejemplo:

function func1(a, b, c) {
    console.log(arguments[0]);
    // Salida: 1

    console.log(arguments[1]);
    // Salida: 2

    console.log(arguments[2]);
    // Salida: 3
}

func1(1, 2, 3);

This

El valor de this está determinado por cómo se invoca a la función. No puede ser establecida mediante una asignación en tiempo de ejecución, y puede ser diferente cada vez que la función es invocada

Ejemplo:

const test = {
    prop: 42,
    func: function() {
        return this.prop;
    },
};

console.log(test.func());
// Salida: 42

Bind

El método bind() crea una nueva función, que cuando es llamada, asigna a su operador this el valor entregado, con una secuencia de argumentos dados precediendo a cualquiera entregados cuando la función es llamada.
El valor de this es ignorado cuando la función es llamada con el operador new.

Ejemplo:

this.x = 9;
let module = {
    x: 81,
    getX: function() { return this.x; }
};

module.getX(); // 81

let getX = module.getX;
getX(); // 9, porque en este caso, "this" apunta al objeto global

// Crear una nueva función con 'this' asociado al objeto original 'module'
let boundGetX = getX.bind(module);
boundGetX(); // 81

Apply

Invoca una determinada función asignando explícitamente el objeto this y un array o similar como parámetros (argumentos) para dicha función.

Ejemplo:

const person = {
    fullName: function() {
        return this.firstName + " " + this.lastName;
    }
}

const person1 = {
    firstName: "Mary",
    lastName: "Doe"
}

// Devuelve "Mary Doe":
person.fullName.apply(person1);

Call

Llama a una función con un valor dado this y con argumentos provistos individualmente..

Ejemplo:

function add(x, y) {
    return x + y;
}

let result = add.call(this, 10, 20);
console.log(result); // 30

Arrow

Llama a una función con un valor dado this y con argumentos provistos individualmente.

Ejemplo:

let a = [
    'Hydrogen',
    'Helium',
    'Lithium',
    'Beryllium'
];
let a3 = a.map(s => s.length);

console.log(a3); // logs [8, 6, 7, 9]
